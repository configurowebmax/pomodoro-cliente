# Timer Pomodoro para Facturar a Clientes Gratis

Aplicación web gratuita con **temporizador Pomodoro** y registro de tiempo facturable por cliente. Mide tu productividad y horas trabajadas de forma sencilla.

## ✨ Características

- Temporizador **Pomodoro** (25 min enfoque / 5 min descanso).
- Contador de **sesiones completadas**.
- Registro de **sesiones facturables** por nombre de cliente.
- Persistencia local (LocalStorage).
- Diseño Mobile-First optimizado.

## 🚀 Uso

1. Pulsa **"Iniciar"** para empezar un bloque de enfoque de 25 minutos.
2. Al terminar, descansa 5 minutos.
3. Usa el formulario para **registrar la sesión** a nombre de un cliente.
